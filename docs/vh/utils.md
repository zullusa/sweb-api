sshOn
-----

метод

описание

Включение доступа по SSH

возвращаемое значение

1 - успешно, 0 - ошибка

параметры в запросе

period

integer

3, 8, 24 часа

тип данных в ответе

integer

пример запроса

{

"jsonrpc":

"2.0"

"method":

"sshOn"

"params":{

"period":

"24"

}

}

пример ответа

{

"jsonrpc":

"2.0"

"id":

"848439544091753.lzizKZZZCS"

"result":

1

}
