# VPS / Мониторинг

Monitoring
==========

объект

описание

Объект сервиса Мониторинг

применяемость

VPS

путь к объекту

POST https://api.sweb.ru/monitoring

требование к авторизации

да

список методов

*   enable;
*   disable;
*   change;
*   plans.

enable
------

метод

описание

Подключение тарифа

возвращаемое значение

1 - тариф успешно подключен, 0 - подключение не удалось

параметры в запросе

id

integer

ID тарифа

тип данных в ответе

integer

пример запроса

```json
{

"jsonrpc":

"2.0"

"method":

"enable"

"params":{

"id":

1

}
```

}

пример ответа

```json
{

"jsonrpc":

"2.0"

"id":

"265139879336589.ltPSelEGPI"

"result":

1

}
```
